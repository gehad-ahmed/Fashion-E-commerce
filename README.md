# 👗 Fashion E-Commerce Website

A responsive fashion e-commerce front-end with a modern storefront layout — hero, product showcase, categories and more.

## 📖 Overview

A fully responsive landing/storefront for a fashion brand, built from scratch with HTML, CSS and JavaScript. It focuses on clean layout, modern styling and adaptability across screen sizes.

## ✨ Features

- 🏬 Modern storefront layout with hero and product sections
- 🧭 Responsive navigation
- 🖼️ Product showcase grid
- 📱 Fully responsive across desktop, tablet and mobile
- ⚡ Lightweight interactions with vanilla JavaScript

## 🛠️ Tech Stack

- HTML5 & CSS3
- Vanilla JavaScript (ES6+)

## 🚀 Getting Started

```bash
git clone https://github.com/gehad-ahmed/Fashion-E-commerce.git
cd Fashion-E-commerce
```

Then open `index.html` in your browser (or use the VS Code **Live Server** extension).
